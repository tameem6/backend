'/api/user/register'
POST. Register new users. Fields - username, name, password
'/api/user/login'
POST. User login. Fields - username, password
'/posts'
GET. Pass auth-token header received on login. Sends all posts.
'/posts'
POST. Pass auth-token header. Fields - In raw format JSON - title and description
'posts/:postId'
GET. Pass auth-token header. Returns post matching postId
'/posts/:postId'
DELETE. Pass auth-token header. Deletes post matching postId
